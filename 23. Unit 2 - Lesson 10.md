## Introduction to Numbers
Q1. 
```
# find quotient using '/' and '//' like (a/b) and (a//b)
a=int(input("a: "))
b=int(input("b: "))
print(a,'/',b,'=',a/b)
print(a,'//',b,'=',a//b)
print(a,'%',b,'=',a%b)
```

## Number Type Conversion
Q1. 
```
# Type Conversion Example
a = '62'
print("converting to int:",int(a),",data type:",type(int(a)))
print("converting to float:",float(a),",data type:",type(float(a)))
print("converting to complex:",complex(a),",data type:",type(complex(a)))
```
Q2. 
```
a=int(input("x: "))

print("Hexadecimal:",hex(a))
```
